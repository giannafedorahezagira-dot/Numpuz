# NumPuz

A modern and interactive number puzzle game built to challenge logic, concentration, and problem-solving skills.

---

## 📖 Description

NumPuz is a puzzle game where players arrange shuffled numbers into the correct order.  
The project focuses on clean UI, smooth gameplay, and responsive design for an engaging user experience.

---

## ✨ Features

- 🎮 Interactive puzzle gameplay
- 🔢 Number tile movement system
- 🧠 Logic and problem-solving challenge
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Clean user interface

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
